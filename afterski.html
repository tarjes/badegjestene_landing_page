Typisk afterski-dag

Eksempel: 3 timer fest-musikk

20:00–20:30 Oppsett og soundcheck
20:30–21:45 Set 1: Klassikere og party-banger
21:45–22:00 Pause – refill drikke!
22:00–23:00 Set 2: Hit-mix, allsang, og DINE favoritter

FEST-PAKKE: Fra 18.000 NOK for 120 min (5 musikere, full energi). PA-system og tekniker pålegges som standard for fest.

  • Musikerer: 5 stk – vokal, gitar, bass, trommer, keys
  • PA + Tekniker: Inkludert i fest-pakken
  • Lengde: Standard 90–120 min, tilpasses
  • Energi-garanti: MÅ få alle på beina!
  • Tilpasset lokasjon: Afterski, utebaren, eller hallen

FAQ: Afterski og Fest

Når må vi booke?

4–6 uker før festen. Afterski-sesongen (jan–apr)? Bo tidlig! Sommerfest? 2–3 måneder før. Vi er fleksible – ta kontakt!

Kan vi sende musikk-ønsker?

YES! Send liste over favoritter, og vi bygger sett rundt det. Your fest = your music!

Hvor stor scen trenger vi?

3x3 meter minimum. Mindre? Vi tilpasser oss. Større? Enda bedre! Outdoor? Vi klarer det!

PA-system – hva skal vi gjøre?

Vi leverer profesjonell PA + tekniker (pa nøkkel). Eller du leverer selv. Avtal det før booking!

Kan vi forhandle på pris?

Alltid! Kortere sett, færre musikere, eller du leverer PA selv – alt påvirker. Snakk med oss!

Spiller dere hele natten?

Gjerne! Lengre sett øker kostnaden naturlig, men vi kan arrangeres!
